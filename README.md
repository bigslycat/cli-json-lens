# jlens [![Build Status][status-img]][status-url]

CLI for getting values from JSON

## Install

- npm: `npm i -g jlens`
- yarn: `yarn global add jlens`

## Usage

```sh
jl package.json dependencies.express
# =>> ^4.16.3
```
